## Conventions

- Spec filenames are kebab-case feature names.
- Every spec has a "Done When" section. The sum of all Done-When sections
  must imply the project-level Done-When in `AGENTS.md`.
- Cross-link specs from `docs/PLANS.md` whenever they map to a milestone.
